
              指纹考勤系统(1.8标准版)说明

    欢迎使用中控指纹考勤系统！本文件向您提供关于该系统的
简单说明。请认真阅读本文件以帮助您正确安装本系统。

一、系统运行需求：

    Microsoft Windows 98 SE 或
    Microsoft Windows 2000

    32M 或更多内存(RAM)

    通用串行总线接口(USB Port)

    U.are.U2000/ YL2000 指纹扫描器

    Pentium 166MHz 或更好的CPU

    CD-ROM驱动器（CD-ROM安装时需要） 

二、安装U.are.U 指纹扫描器及其指纹考勤系统软件

1、安装指纹扫描器
   如果您还没有在您的计算机上安装指纹扫描器驱动程序，安
装程序会要求您安装该驱动程序。
 
   在安装指纹扫描器驱动程序前，如果您已经把指纹扫描器连
接到计算机上，请断开联线。
   设备驱动程序安装前，您需要确保计算机的USB接口工作正常，
并且能够在“控制面板”->“系统”->“设备管理器”中看到
“通用串行总线控制器” 的至少“USB Root Hub”和
“...Universal Host Controller”两个设备。

    指纹扫描器在安装设备驱动程序前不能连接到计算机上，
在安装过程快要结束时，请根据提示把指纹识别器连接到计算
机USB接口上。
    在安装过程中，可能会要求您插入Windows 95/98光盘，安
装程序提取到所需Windows文件后，您需要重新开始安装程序。
    指纹扫描器驱动程序安装完成后，镜头的红光灯
就会点亮。

 
2、安装指纹考勤系统
    运行 CDROM 根目录上setup.exe程序。
    选择要安装指纹考勤系统的目录，安装程序将会把程序文
件和数据文件安装到该目录下，并在计算机桌面上自动建立程
序的快捷方式。

三、登记注册

    为了得到最新产品技术信息和取得技术支持，请您登记注
册。阅读register.txt 文件以获取注册指导。

四、安装盘说明

setup.exe 	指纹考勤系统安装程序
readme.txt	本文件
register.txt	软件注册登记表
driver\		指纹扫描器驱动程序
mdac\		Microsoft 数据库支持
