
         指紋差勤系統說明

    歡迎使用指紋差勤系統！本檔案向您提供關于該系統的
簡單說明。請認真閱讀本檔案以幫助您正確安裝本系統。

一、系統運行需求：

    Microsoft Windows 98(SE) 或
    Microsoft Windows 2000

    32M 或更多內存(RAM)

    通用串列匯流接口(USB Port)

    U.are.U2000/ YL2000 指紋掃描器

    Pentium 166MHz 或更好的CPU

    CD-ROM驅動器（CD-ROM安裝時需要） 

二、安裝U.are.U 指紋掃描器及其指紋差勤系統軟體

1、安裝指紋掃描器
   如果您還沒有在您的計算機上安裝指紋掃描器驅動程式，安
裝程式會要求您安裝該驅動程式。
 
   在安裝指紋掃描器驅動程式前，如果您已經把指紋掃描器連
接到計算機上，請斷開聯線。
   設備驅動程式安裝前，您需要確保計算機的USB接口工作正常，
并且能夠在“控制台”->“系統”->“設備管理器”中看到
“通用串列匯流排” 的至少“USB Root Hub”和
“...Universal Host Controller”兩個設備。

    如果安裝程式要求您輸入一個16位的加密碼值，請輸入
“1234567890123456”即可。

    指紋掃描器在安裝設備驅動程式前不能連接到計算機上，
在安裝過程結束時，請把指紋辨識器連接到計算機USB接口
上，然后根據提示進行操作。
    在安裝過程中，可能會要求您插入Windows 95/98光盤，安
裝程式提取到所需Windows檔案后，您需要重新開始安裝程式。
    指紋掃描器驅動程式安裝完成后，鏡頭的紅光燈就會點亮。

 
2、安裝指紋差勤系統
    運行 CDROM 根目錄上setup.exe程式。
    選擇要安裝指紋差勤系統的目錄，安裝程式將會把程式文
件和數據檔案安裝到該目錄下，并在計算機桌面上自動建立程
序的快捷方式。

三、登記注冊

    為了得到最新產品技朮訊息和取得技朮支援，請您登記注
冊。閱讀register.txt 檔案以獲取注冊指導。
