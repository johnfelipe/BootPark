Instruction to Wiegand input configuration

Wiegand input format: Assigns Wiegand inputs format, following detailed introduction.

Wiegand input the total bit : default 26 bit 

Wiegand pulse width time: Assigns pulse width time , default 100μm

Wiegand pulse interval times : Assigns pulse interval times , default 1000μm

About Wiegand input format,

Wiegand input format include: OEM code (for 26bit-wiegand generally do not need to be defined, find it from 37, 34bit-Wiegand), Facility code (machine number or refers to site code), ID Number (user serial number), its first letter respectively indicate (no matter it is in capital letter or small letter) in the form, ofiOFI, the small letter express odd (odd), the capital letter refers to even (Even), Oo refers to OEM code, Ff refers to Facility code, Ii is ID number, P express the parity bit, following is demonstration with a standard 26bit-Wiegand input, the definition: PFFFFFFFFIIIIiiiiiiiiiiiiP, Facility code is 1, ID Number also is 1 input  :
